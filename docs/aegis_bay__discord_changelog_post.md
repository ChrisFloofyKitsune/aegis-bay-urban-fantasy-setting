# :memo: Aegis Bay — Quick Changelog (v0.5 → v0.6)

**Setting revision focus:** broader sandbox, same strong bones.

* :arrows_counterclockwise: **Wider story support.** The setting now actively makes room for slice-of-life, coming-of-age, romance, nightlife, and everyday-errand stories alongside the procedural drama, labor grit, and occult mystery lanes that already worked. Same city, more kinds of lives.
* :round_pushpin: **Locations refreshed.** Added **The Promenade** (the commercial mall district, now home to Lumina Heroics HQ). District genre tags broadened. Canon POIs back in the city guide.
* :bus: **New post above:** a dedicated Stop Menu listing every story lane you can launch from the Route 42 bus.
* :wind_blowing_face: **Magic feels more like the Bay.** Sensory cues now lean into coastal weather — humidity pressing against skin, brine and wet stone, fog clinging slower, the bay feeling closer — instead of the same generic defaults.
* :scroll: **Open source.** The full world prompt and design archive are now public on GitHub under CC BY 4.0: <https://github.com/ChrisFloofyKitsune/aegis-bay-urban-fantasy-setting>

Pitch, locations, and stop-menu posts above have all been updated to match.
